Clipto 提供了一种全新的方式来帮助您组织您的日常事务。重要或是临时的笔记？交给可靠的 Clipto 吧，它会既快速又方便地帮您做好一切。


【在 Mac、Windows、Linux 及浏览器间同步】
试用网页版 Clipto，并下载桌面版程序，请访问官网 https://clipto.pro
Clipto 可以使所有内容保持同步。


【支持离线使用】
每一项功能都可以完全离线运行。 如果您想让您的笔记跨所有设备同步，仅需登录同一账号即可。 如果不需要，则无需登录，但仍可以使用程序的手动导入/导出功能。 这一切完全由您自己做主。


【从剪贴板内容创建笔记】
Clipto 可作为一款剪贴板管理器，甚至是在有着剪贴板限制的 Android 10 上。


【激活通用剪贴板】
使用通用剪贴板，您可以在一台设备上复制笔记，然后将内容粘贴到另外一台设备上。


【扫描条形码】
使用 Clipto，您可以使用相机创建笔记并将其保存到应用中。
设备上的条形码扫描功能可读取大多数标准格式(Codabar，Code 39，Code 93，Code 128，EAN-8，EAN-13，ITF，UPC-A，UPC-E，Aztec，Data Matrix，PDF417，QR Code)。


【添加文档】
在 Clipto 中，您可以保存任何文档（不仅仅是图像或照片）


【个性化】
安装完成后，Clipto 允许您根据需要自定义任何细微的小功能：
• 主题配色（白色、复古色、绿色、夜间、深蓝色、夜间蓝、A屏黑）
• 列表样式（网格、自适应、简洁、预览）
• 排序方式（创建日期、使用日期、使用次数、标题、文本、标签、附件大小、字数）
• 文字大小（列表项和笔记）
• 文字字体（列表项和笔记）
• 自定义列表滑动手势


【分享】
为笔记创建公开链接，无需注册即可在网络上阅览。
公开链接可包含以下属性：
• 访问密码
• 一次性访问
• 访问时间
• 到期时间


【提高您的生产力】
Clipto 生而易用，提供了各种各样创建笔记的方式：
✔ 通知操作 - 访问最近的笔记，支持一键式操作；
✔ 弹出菜单操作 - 在任一应用中，选中文本，长按弹出上下文菜单以找到扩展功能；
✔ 剪贴板中的笔记 - 保存剪贴板中的笔记，后续还可以当做剪贴板历史记录；
✔ 剪贴板操作 - 从剪贴板中提取所需的笔记，并一键式操作；
✔ 分享操作 - 从分享处接收到的任何文本或文件中创建笔记；
✔ 应用内编辑器 - 以纯文本，Markdown，HTML 格式创建笔记；
✔ 一键式操作
    • 文字转语音
    • 撰写电子邮件
    • 撰写短信
    • 撰写推文
    • 添加日历活动
    • 插入联系人
    • 网络搜索
    • 维基百科搜索
    • YouTube 搜索
    • Google 翻译
    • 分享给…
    • 复制到剪贴板
    • 以二维码发送
    • 以 Txt 发送
    • 以 Pdf 发送
    • 以 Markdown 发送
    • 打开网页
    • 保存联系人电子邮件
    • 保存联系电话
    • 拨打电话


【从其他笔记应用和剪贴板管理器导入】
有人希望能够从一些主流的应用中导入笔记。 所以我们做了这个功能！现在正式支持以下备份格式：
• Clipper + JSON
• Simplenote JSON
• Clip Stack
• Clipboard Manager
• Google Keep（zip 格式）
