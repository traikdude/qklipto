Clipto는 빠르고 편리하고 안정적으로 작업해야 할 때 중요한 노트나 임시 노트로 일과를 정리할 수 있는 새로운 방법입니다.


맥, 윈도우, 리눅스, 브라우저와 동기화하세요
웹에서 Clipto를 써보고 https://clipto.pro에서 데스크탑용 앱을 내려받으세요.
Clipto는 모든 것을 동기화해둡니다.


오프라인 작업
각 기능은 오프라인에서도 완벽하게 쓸 수 있습니다. 노트를 모든 기기에 동기화하고 싶을 때는, 그냥 같은 계정으로 로그인하시면 됩니다. 그렇지 않다면, 로그인하지 않고 수동으로 내보내기/가져오기 기능을 이용하시면 됩니다. 모든 것은 당신이 결정합니다.


클립보드로부터 노트 가져오기
Clipto를 클립보드 관리자로 사용하실 수 있습니다. 심지어 Android 10에서도요!


전역 클립보드 활성화
전역 클립보드가 있다면 한 기기에서 노트를 복사하고, 그것을 다른 기기에 붙여넣을 수 있습니다.


바코드 스캔
Clipto를 사용하면 카메라를 통해 노트를 촬영하고 그걸 앱에 저장할 수 있습니다.
바코드 스캔 기능은 대부분의 표준 포맷(Codabar, Code 39, Code 93, Code 128, EAN-8, EAN-13, ITF, UPC-A, UPC-E, Aztec, Data Matrix, PDF417, QR Code)을 지원합니다.


파일 추가
Clipto에는 사진뿐만이 아닌 어떤 파일이든 저장할 수 있습니다.


개인화
Clipto에서는 필요한 경우 어떤 사소한 동작이든 사용자 지정할 수 있습니다.
• 테마 (흰색, 세피아, 녹색, 어둡게, 짙은 파랑, 보랏빛 군청색, 완전 검정)
• 목록 스타일 (격자, 널널하게, 빽빽하게, 미리보기)
• 조건을 통해 정렬하기 (생성일, 사용일, 사용 횟수, 제목, 내용, 태그, 크기, 글자 수)
• 글씨 크기 (목록 및 노트 화면 모두)
• 글씨체 (목록 및 노트 화면 모두)
• 목록에서 스와이프 동작


공유
노트의 공용 링크를 생성한 다음 노트를 가입없이 웹에서 접근할 수 있도록 해보세요.
공용 링크의 속성은 아래 속성들의 모든 조합으로 구성할 수 있습니다.
• 사용자가 지정한 비밀번호
• 1회만 접근 허용
• 접근 지연
• 유효 기간


생산성을 증대시키세요
Clipto는 간단함과 노트 작성 시 사용할 수 있는 모든 기능을 갖추도록 디자인되었습니다.
✔ 알림 동작 - 최근 노트에 빠르게 접근한 다음 한 번의 터치로 관리하세요
✔ 상황에 맞는 동작 - 애플리케이션에서 텍스트를 선택할 때 여러 기능에 빠르게 접근할 수 있도록 합니다.
✔ 클립보드 노트 - 클립보드를 노트로 저장한 이후 클립보드 기록으로 사용할 수 있습니다.
✔ 클립보드 동작 - 클립보드에서 노트의 일부를 추출한 다음 단 한 번의 터치로 사용하세요.
✔ 공유 동작 - 공유한 텍스트나 파일에서 노트를 생성합니다.
✔ 앱 내 편집기 - 평문, 마크다운, HTML 등으로 노트를 작성하세요.
✔ 한 번 터치로 동작
    • 텍스트 음성 합성

    • YouTube 검색
    • 구글 번역
    • 공유하기…

다른 노트 앱 및 클립보드 매니저의 데이터 가져오기
사용자 여러분들이 일부 유명 앱들에서 노트를 가져오는 기능을 요청하셨습니다. 그래서 기능을 개발했고 현재는 아래 유형의 데이터를 공식적으로 지원합니다:
• Clipper+ JSON
• Simplenote JSON
• 클립 스택
• 클립보드 관리자
• 구글 킵 (ZIP 형식의 데이터)
