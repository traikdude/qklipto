Clipto可以以快速，方便且安全的方式紀錄筆記，在需要的時候，安排您的日常工作。


與MAC、WINDOWS、LINUX和瀏覽器同步
試試Clipto的網頁版，並從官網 https://clipto.pro下載桌面應用程式。
Clipto使所有內容保持同步。


離線工作
每個操作都可以完全離線工作。 若要在所有裝置上同步筆記，只需登入相同帳戶即可。 您也可以選擇不登入而手動使用匯出/導入功能。 這完全由您决定。


從剪貼簿記錄筆記
Clipto甚至可以在Android 10上工作！


啟用通用剪貼簿
使用通用剪貼簿，您可以在當前裝置中複製筆記，並將內容黏貼到另一個裝置上。


掃描條碼
Clipto允許您使用相機捕捉筆記並將其保存到應用程式中。
裝置上的條碼掃描可讀取大多數標準格式 (Codabar, Code 39, Code 93, Code 128, EAN-8, EAN-13, ITF, UPC-A, UPC-E, Aztec, Data Matrix, PDF417, QR Code)。


新增檔案
在Clipto中，您可以儲存任何類型的檔案 (不僅僅是圖像或照片)


個性化
安裝完成後，Clipto允許您根據需要自訂任何最小行為：
•主題(白色，復古，綠色，粉色，夜間，深藍色，夜間藍，AMOLED黑色)
•列表樣式(網格，自適應，精簡，預覽)
• 排序方式(創建日期，編輯日期，使用日期，使用次數，標題，文本，標籤，檔案大小，字數)
•文字大小(適用於列表和筆記)
• 文字字體(適用於列表和筆記)
•在主列表中的滑動操作


分享
在筆記上新增一個公開連結，使其無需註冊即可在瀏覽器中訪問。
公開連結可以具有以下內容的任意組合:
• 使用者自訂密碼
• 單次訪問
• 延遲訪問
• 到期時間


提升您的效率
Clipto的設計在於簡化操作，並提供了所有可用於記錄筆記的方法:
✔通知操作 - 快速訪問最新筆記並進行一鍵式操作；
✔文本選擇 - 在任何應用程式中透過長按所選文本訪問Clipto的擴展功能；
✔複製到剪貼簿的筆記 - 儲存複製到剪貼簿的筆記，日後可以在歷史記錄中查看並使用；
✔剪貼簿操作-從剪貼簿中提取筆記，並與其他操作同時使用;
✔分享操作-從任何分享的文字或檔案捕獲筆記;
✔ 應用內編輯器 - 以純文本，Markdown，HTML創建筆記;

從其他筆記和剪貼簿應用程式導入數據
您可以要求Clipto新增導入其他熱門應用程式的筆記。 是的，我們目前正式支持以下類型的備份格式：

• Clip Stack
• 剪貼簿管理
• Google Keep (takeout zip)
