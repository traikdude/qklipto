 ## 9. Security Considerations

 ### 9.1 Local Mode Security
 | Risk | Mitigation |
 |------|------------|
 | LAN exposure | Server binds to      
 `0.0.0.0` but only on local network | 
 | No authentication | Acceptable for  
 personal single-user use |
 | Data in plain JSON | Keep `db.json` 
  in user-owned directory |

 ### 9.2 Cloud Mode Security
 | Risk | Mitigation |
 |------|------------|
 | Unauthorized access | Firebase Auth 
  + Firestore security rules |
 | Data at rest | Firebase encrypts by 
  default |
 | API key exposure | Keys are safe to 
  expose (security rules enforce       
 access) |

 ### 9.3 Desktop App Security
 | Risk | Mitigation |
 |------|------------|
 | Electron node integration | Use     
 preload script with contextBridge |   
 | XSS attacks | React escapes by      
 default; DOMPurify for rich text |    
 | Local data access | Stored in       
 user's AppData (OS-protected) |
